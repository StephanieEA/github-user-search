# Github User Search
Deployed at: https://github-user-search-sea.herokuapp.com/

Github is a personal project built with React to search Github users 

## Running Locally

Clone the repositiory

Install dependencies
```bash
npm install
```
Run the development server with 
```bash
npm run start
```
